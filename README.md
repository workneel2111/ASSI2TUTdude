# LaundryLux Project

A simple, responsive landing page for a laundry service using HTML and CSS.

## How to Run

1.  **Open locally:** Locate the `index.htm` file in your file explorer and double-click it to open it in your default web browser.
2.  **Using Live Server:** If you are using VS Code, right-click `index.htm` and select **"Open with Live Server"**.

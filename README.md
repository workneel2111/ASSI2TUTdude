# LaundryLux Project

This is a landing page for "LaundryLux," a laundry service company. I built this project to practice my HTML, CSS, and JavaScript skills.

## Features Implemented
- **Responsive Design**: Used a mobile-first approach with media queries for desktop layouts.
- **Dynamic Cart**: Users can add or remove laundry services, and the total price updates automatically.
- **Smooth Scrolling**: The booking button takes you directly to the services section.
- **Email Integration**: Integrated EmailJS to send booking details from the form to an email address.
- **Interactive UI**: Hover effects on buttons and icons for better user experience.

## Challenges Faced
- **Cart Logic**: It was tricky to figure out how to remove only one instance of an item at a time. I ended up using a loop to find the last occurrence of an item ID.
- **Responsive Layout**: Switching from a single column on mobile to a multi-column flexbox layout on desktop took some trial and error with CSS breakpoints.

## How to Run
1. Clone or download the repository.
2. Open `index.htm` in any modern web browser.
3. Add services to your cart and fill out the booking form to see it in action!
